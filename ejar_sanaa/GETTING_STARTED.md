# دليل البدء السريع - إيجار صنعاء

## المتطلبات الأساسية

- Flutter SDK 3.0 أو أحدث
- Dart SDK 3.0 أو أحدث
- Android Studio / VS Code مع Flutter Extension
- جهاز أندرويد / iOS أو محاكي

## خطوات التشغيل

### 1. تثبيت التبعيات

```bash
cd ejar_sanaa
flutter pub get
```

### 2. تشغيل التطبيق

```bash
flutter run
```

### 3. إنشاء تطبيق للأندرويد

```bash
flutter build apk --release
```

### 4. إنشاء تطبيق لـ iOS (فقط على macOS)

```bash
flutter build ios --release
```

## هيكل المشروع

```
ejar_sanaa/
├── lib/
│   ├── core/
│   │   ├── constants/       # الثوابت (المديريات، الأنواع)
│   │   └── theme/           # الثيم والألوان
│   ├── models/              # نماذج البيانات
│   ├── providers/           # إدارة الحالة
│   ├── screens/             # الشاشات
│   ├── widgets/             # العناصر القابلة لإعادة الاستخدام
│   └── main.dart            # نقطة البداية
├── pubspec.yaml             # التبعيات والإعدادات
└── README.md
```

## الميزات الرئيسية

✅ دعم كامل للغة العربية (RTL)  
✅ 10 مديريات في أمانة العاصمة  
✅ أنواع إيجار متعددة (شقق، فلل، سيارات، إلخ)  
✅ فلاتر ذكية (السعر، الموقع، الشمس، الماء، الكهرباء)  
✅ تصميم مناسب للثقافة اليمنية  
✅ واجهة سهلة الاستخدام للمراهقين  

## ملاحظات مهمة

- التطبيق حالياً يستخدم بيانات تجريبية (Mock Data)
- ستحتاج إلى إضافة Backend API لربط التطبيق بخادم حقيقي
- يمكنك توسيع قائمة الأحياء والحارات في `districts.dart`
- الألوان مستوحاة من العمارة الصنعانية (الطوب الأحمر، الجص الأبيض)
